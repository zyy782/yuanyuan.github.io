### 功能描述

1. 引入组件，并绑定点击事件 <br>``` <change @click.native="changeSkin"></change>```
2. 具体方法<br>
   定义变量isActive，为true执行换肤，false回到初始皮肤
   ```  changeSkin() {
      if (this.isActive) {
        this.$refs.home.style.backgroundImage =
          "linear-gradient(#6d78b7, #2ebf91)"
        this.isActive = false
      } else {
        this.$refs.home.style.backgroundImage = "none"
        this.isActive = true
      }
    },
    ```
