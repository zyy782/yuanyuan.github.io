## Nodecat 性能监控
`Nodecat` 是一个 Node.js 客户端库，用于与 [Cat](https://github.com/dianping/cat) 系统进行交互。Cat 是一个分布式实时监控系统，主要用于记录和分析应用程序的性能数据、错误日志和业务指标。详细可以看官方仓库：https://github.com/dianping/cat

### `nodecat` 的主要功能
1. **Transaction**: 用于记录一段代码的执行时间和状态。
2. **Event**: 用于记录单个事件的状态和信息。
3. **Error**: 用于记录带有堆栈信息的错误。

### 使用场景
- **性能监控**: 记录代码执行时间，分析性能瓶颈。
- **错误追踪**: 捕获异常和错误堆栈，便于调试。
- **业务指标**: 记录业务相关的事件和数据，生成报表。

### 示例代码
```javascript
const cat = require('@dp-cat/client');

// 初始化
cat.init({
    appkey: 'your-appkey'
});

// 创建 Transaction
let t = cat.newTransaction('Context', 'TransactionName');
try {
    // 业务逻辑
    t.setStatus(cat.STATUS.SUCCESS);
} catch (error) {
    t.setStatus(cat.STATUS.FAIL);
    cat.logError('TransactionError', error);
} finally {
    t.complete();
}

// 记录事件
cat.logEvent('EventType', 'EventName', cat.STATUS.SUCCESS, 'EventData');

// 记录错误
cat.logError('ErrorType', new Error('Something went wrong'));
```

### 特点
- 支持嵌套的 Transaction 和 Event。
- 提供多种 API 来记录数据和状态。
- 支持将错误堆栈信息存储到 Cat 系统中，便于调试。

### 依赖
`nodecat` 需要安装 `libcatclient.so`，并将其放置在 `LD_LIBRARY_PATH` 中。
